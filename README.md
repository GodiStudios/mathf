# mathf

Math basics for a game frameworks and engines.
[![Build Status](https://travis-ci.org/GodiStudios/mathf.svg)](https://travis-ci.org/GodiStudios/mathf)

* 2D Vector
* 3D Vector
* 3 x 3 Matrix
* 2 x2 Matrix
